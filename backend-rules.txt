# REGLAS BACKEND - Laravel 12.0

## STACK TECNOLOGICO

- Framework: Laravel 12.0
- Base de datos: PostgreSQL con UUIDs
- ORM: Eloquent
- Autenticacion: JWT (tymon/jwt-auth 2.x)
- Validacion: Form Requests
- API: RESTful con recursos anidados
- Servidor: PHP 8.3+
- Composer para dependencias

### Codigo

- PSR-12 para formato de codigo
- 4 espacios para indentacion (no tabs)
- Type hints obligatorios en metodos
- Return types obligatorios
- DocBlocks en metodos publicos
- Tipos estrictos (strict_types declare)
- Nullable types con ?
- named arguments para parametros opcionales

```php
<?php

declare(strict_types=1);

namespace App\Http\Controllers\Api\V1;

use App\Http\Controllers\Controller;
use App\Http\Resources\ProductResource;
use App\Repositories\ProductRepository;
use Illuminate\Http\JsonResponse;
use Illuminate\Http\Request;

/**
 * Controlador de productos
 */
class ProductController extends Controller
{
    public function __construct(
        private readonly ProductRepository $productRepository
    ) {}

    /**
     * Listar productos con paginacion
     */
    public function index(Request $request): JsonResponse
    {
        $products = $this->productRepository->paginate(
            perPage: $request->integer('per_page', 15),
            page: $request->integer('page', 1)
        );

        return response()->json([
            'data' => ProductResource::collection($products),
            'meta' => [
                'current_page' => $products->currentPage(),
                'last_page' => $products->lastPage(),
                'per_page' => $products->perPage(),
                'total' => $products->total(),
            ],
        ]);
    }
}
```

## PATRONES

### Repository Pattern

Interfaz del repositorio:

```php
// app/Repositories/Contracts/ProductRepositoryInterface.php
<?php

namespace App\Repositories\Contracts;

use App\Models\Product;
use Illuminate\Database\Eloquent\Collection;
use Illuminate\Pagination\LengthAwarePaginator;

interface ProductRepositoryInterface
{
    public function findByUuid(string $uuid): ?Product;
    public function paginate(int $perPage = 15, int $page = 1): LengthAwarePaginator;
    public function getByCategory(string $categoryUuid): Collection;
    public function search(string $term): Collection;
    public function create(array $data): Product;
    public function update(Product $product, array $data): Product;
    public function delete(Product $product): bool;
}
```

Implementacion del repositorio:

```php
// app/Repositories/ProductRepository.php
<?php

namespace App\Repositories;

use App\Models\Product;
use App\Repositories\Contracts\ProductRepositoryInterface;
use Illuminate\Database\Eloquent\Collection;
use Illuminate\Pagination\LengthAwarePaginator;
use Illuminate\Support\Facades\DB;

class ProductRepository implements ProductRepositoryInterface
{
    public function findByUuid(string $uuid): ?Product
    {
        return Product::where('uuid', $uuid)->first();
    }

    public function paginate(int $perPage = 15, int $page = 1): LengthAwarePaginator
    {
        return Product::with('category')
            ->orderBy('created_at', 'desc')
            ->paginate($perPage, ['*'], 'page', $page);
    }

    public function getByCategory(string $categoryUuid): Collection
    {
        return Product::whereHas('category', function ($query) use ($categoryUuid) {
            $query->where('uuid', $categoryUuid);
        })->get();
    }

    public function search(string $term): Collection
    {
        return Product::where('name', 'ilike', "%{$term}%")
            ->orWhere('description', 'ilike', "%{$term}%")
            ->get();
    }

    public function create(array $data): Product
    {
        return DB::transaction(function () use ($data) {
            return Product::create($data);
        });
    }

    public function update(Product $product, array $data): Product
    {
        $product->update($data);
        return $product->fresh();
    }

    public function delete(Product $product): bool
    {
        return $product->delete();
    }
}
```

### Service Layer

```php
// app/Services/ProductService.php
<?php

namespace App\Services;

use App\Models\Product;
use App\Repositories\Contracts\ProductRepositoryInterface;
use Illuminate\Database\Eloquent\Collection;
use Illuminate\Pagination\LengthAwarePaginator;
use Illuminate\Support\Str;

class ProductService
{
    public function __construct(
        private readonly ProductRepositoryInterface $productRepository
    ) {}

    public function getAll(int $perPage = 15, int $page = 1): LengthAwarePaginator
    {
        return $this->productRepository->paginate($perPage, $page);
    }

    public function getByUuid(string $uuid): ?Product
    {
        $product = $this->productRepository->findByUuid($uuid);

        if (!$product) {
            throw new \Exception('Producto no encontrado', 404);
        }

        return $product;
    }

    public function create(array $data): Product
    {
        $data['uuid'] = Str::uuid()->toString();

        return $this->productRepository->create($data);
    }

    public function update(string $uuid, array $data): Product
    {
        $product = $this->getByUuid($uuid);

        return $this->productRepository->update($product, $data);
    }

    public function delete(string $uuid): bool
    {
        $product = $this->getByUuid($uuid);

        return $this->productRepository->delete($product);
    }
}
```

### Form Requests

```php
// app/Http/Requests/StoreProductRequest.php
<?php

namespace App\Http\Requests;

use Illuminate\Foundation\Http\FormRequest;

class StoreProductRequest extends FormRequest
{
    public function authorize(): bool
    {
        return true;
    }

    public function rules(): array
    {
        return [
            'name' => ['required', 'string', 'max:255'],
            'description' => ['nullable', 'string', 'max:1000'],
            'price' => ['required', 'numeric', 'min:0', 'decimal:0,2'],
            'stock' => ['required', 'integer', 'min:0'],
            'category_uuid' => ['required', 'uuid', 'exists:categories,uuid'],
            'image_url' => ['nullable', 'url', 'max:500'],
            'is_active' => ['boolean'],
        ];
    }

    public function messages(): array
    {
        return [
            'name.required' => 'El nombre es obligatorio',
            'price.min' => 'El precio debe ser mayor a 0',
            'category_uuid.exists' => 'La categoria no existe',
        ];
    }
}
```

### Middleware de Auth

```php
// app/Http/Middleware/JwtAuthenticate.php
<?php

namespace App\Http\Middleware;

use Closure;
use Illuminate\Http\Request;
use Symfony\Component\HttpFoundation\Response;
use Tymon\JWTAuth\Facades\JWTAuth;
use Tymon\JWTAuth\Exceptions\JWTException;
use Tymon\JWTAuth\Exceptions\TokenExpiredException;
use Tymon\JWTAuth\Exceptions\TokenInvalidException;

class JwtAuthenticate
{
    public function handle(Request $request, Closure $next): Response
    {
        try {
            $user = JWTAuth::parseToken()->authenticate();

            if (!$user) {
                return response()->json([
                    'success' => false,
                    'message' => 'Usuario no encontrado',
                ], 401);
            }
        } catch (TokenExpiredException $e) {
            return response()->json([
                'success' => false,
                'message' => 'Token expirado',
            ], 401);
        } catch (TokenInvalidException $e) {
            return response()->json([
                'success' => false,
                'message' => 'Token invalido',
            ], 401);
        } catch (JWTException $e) {
            return response()->json([
                'success' => false,
                'message' => 'Token no proporcionado',
            ], 401);
        }

        return $next($request);
    }
}
```

```php
<?php

namespace App\Models\Traits;

use Illuminate\Support\Str;

trait HasUuid
{
    protected static function bootHasUuid(): void
    {
        static::creating(function ($model) {
            if (empty($model->uuid)) {
                $model->uuid = Str::uuid()->toString();
            }
        });
    }

    public function getRouteKeyName(): string
    {
        return 'uuid';
    }
}
```

## PROHIBICIONES

- NO usar id numerico (siempre UUID)
- NO acceder a modelos directamente en controladores (usar Services)
- NO escribir querys en controladores (usar Repositories)
- NO retornar arrays (usar API Resources)
- NO validar en controladores (usar Form Requests)
- NO usar $_GET, $_POST (usar Request)
- NO hardcodear respuestas JSON (usar recursos)
- NO crear metodos estaticos en modelos
- NO usar DB::raw sin sanitizar
- NO guardar secrets en codigo (usar .env)
- NO retornar Eloquent collections como JSON directamente
- NO validar en vistas (solo en API)
- NO usar Route::resource sin customizar
- NO ignorar errores (usar try-catch en services)

```php
namespace Tests\Feature;

            ->assertJsonFragment(['name' => 'Test Product']);

    {
```

### Ejemplo Test de Service

```php
// tests/Unit/ProductServiceTest.php
<?php

namespace Tests\Unit;

use App\Models\Product;
use App\Repositories\Contracts\ProductRepositoryInterface;
use App\Services\ProductService;
use PHPUnit\Framework\TestCase;
use Mockery;

class ProductServiceTest extends TestCase
{
    protected function tearDown(): void
    {
        Mockery::close();
    }

    public function test_get_all_returns_paginated_products(): void
    {
        $mockRepository = Mockery::mock(ProductRepositoryInterface::class);
        $mockPaginator = Mockery::mock(LengthAwarePaginator::class);

        $mockRepository->shouldReceive('paginate')
            ->with(15, 1)
            ->once()
            ->andReturn($mockPaginator);

        $service = new ProductService($mockRepository);
        $result = $service->getAll();

        $this->assertEquals($mockPaginator, $result);
    }

    public function test_get_by_uuid_returns_product(): void
    {
        $mockProduct = Mockery::mock(Product::class);
        $mockRepository = Mockery::mock(ProductRepositoryInterface::class);

        $mockRepository->shouldReceive('findByUuid')
            ->with('valid-uuid')
            ->once()
            ->andReturn($mockProduct);

        $service = new ProductService($mockRepository);
        $result = $service->getByUuid('valid-uuid');

        $this->assertEquals($mockProduct, $result);
    }
```

### Helper Base64

```php
// app/Utils/Base64Helper.php
<?php

namespace App\Utils;

class Base64Helper
{
    public static function encode(string $uuid): string
    {
        return base64_encode($uuid);
    }

    public static function decode(string $encodedId): string
    {
        $decoded = base64_decode($encodedId);

        if ($decoded === false) {
            throw new \InvalidArgumentException('Invalid base64 encoded ID');
        }

        return $decoded;
    }

    public static function isValid(string $encodedId): bool
    {
        return base64_decode($encodedId, strict: true) !== false;
    }
}
```

### Trait para Base64 en Controller

```php
// app/Http/Controllers/Traits/DecodesBase64Ids.php
<?php

namespace App\Http\Controllers\Traits;

use App\Utils\Base64Helper;

trait DecodesBase64Ids
{
    protected function decodeUuid(string $encodedId): string
    {
        try {
            return Base64Helper::decode($encodedId);
        } catch (\InvalidArgumentException $e) {
            abort(400, 'ID invalido');
        }
    }

    protected function decodeUuids(array $encodedIds): array
    {
        return array_map(fn($id) => $this->decodeUuid($id), $encodedIds);
    }
}
```

## NOTAS

- IDs siempre en base64 para frontend
- Usar decodeUuid() en controladores
- JWT config en config/jwt.php
- PostgreSQL: usar uuid_generate_v4()
- Migraciones: siempre withUuids() trait
- Servicios: throw exceptions, no retornar null en errores
- Repositorios: siempre interfaces
- API Resources: siempre tipar arrays
- Form Requests: validar en orden
- Tests: usar factories, no hardcoded
- Middleware: registrar en Kernel.php
- Rutas: agrupar por version
